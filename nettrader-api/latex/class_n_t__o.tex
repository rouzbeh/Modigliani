\subsubsection{Detailed Description}
It provides object accounting, and a unique identifier (uniqId) for each object. The unique id might become ambigous when uniqId gets greater LONG\_\-MAX. However, the initialisation of each NT object with this class might help in synchronisation of a threaded version.
